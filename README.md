# SCAP
Simpelest CPU Arch Possible

# Installing
To install the emulator and assembler you can simply run the install script like this:
```bash
chmod +x install.sh && ./install.sh
```

# Usage
**qemu-emu**: qemu-emu [-d] <rom>
**qemu-as**: qemu-as <input file>
